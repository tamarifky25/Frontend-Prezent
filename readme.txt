how to use:

+ extract prezent folders
+ open extracted folders ("new-prezent")
+ open cmd or git bash, and type "npm install" command
+ type "npm start" to run locally

prerequisites :
+ Has installed node.js

Change API Endpoint :
To change api endpoint address, please go to ./src/helpers/constants.js, 
and change endpoint constant values according to your endpoint address.

External Libraries Documentation :

+ react-fontawesome : https://fontawesome.com/how-to-use/on-the-web/using-with/react
